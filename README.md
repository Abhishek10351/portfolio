# Personal Portfolio

A modern, responsive portfolio built with **Next.js**, **TypeScript**, and **Tailwind CSS** to showcase my projects, technical skills, and experience as a Full Stack Developer.

> **Status:** 🚧 Currently under development

---

## Overview

This portfolio is designed to be more than a personal website. It serves as a central place to showcase my work, explain the engineering decisions behind each project, and provide recruiters and developers with an overview of my technical skills.

The focus is on clean design, accessibility, performance, and an engaging user experience while keeping the content as the primary highlight.

---

## Goals

* Showcase featured software projects
* Present detailed project case studies
* Highlight technical skills and experience
* Provide downloadable resume and contact information
* Deliver a fast, responsive, and accessible experience
